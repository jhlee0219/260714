# Nintendo.com (2001) inspired design

This project follows the Nintendo.com (2001) design analysis from
[VoltAgent/awesome-design-md](https://github.com/VoltAgent/awesome-design-md/tree/main/design-md/nintendo-2001).

## Applied language

- Periwinkle beveled console-chrome panels
- Carbon command bars with halftone texture
- Amber and signal-orange reserved for controls and forward actions
- Sharp and chamfered geometry with rounded physical controls
- Outlined, hard-shadowed Arial Black display type
- Dense modular layout adapted to stack on narrow screens
